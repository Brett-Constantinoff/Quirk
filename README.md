## Running Quirk

Clone using:

```sh
git clone https://github.com/Brett-Constantinoff/Quirk.git --recursive
```

Run the Setup.bat file by double clicking it, this will generate a visual studio solution file.

## Dependencies
* Glfw
* Glm
* Glad
* Imgui
* Entt
* Assimp
* Spdlog
